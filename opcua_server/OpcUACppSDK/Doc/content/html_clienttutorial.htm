The client tutorial uses the Server Tutorial Step 4 Sample Application  to access the features implemented by the client tutorial. This tutorial shows step by step how to implement an OPC UA client application that uses the most essential features of OPC UA:

•Step 1: Describes the initialization and termination of an OPC UA application and how to establsih a connection to a server.
•Step 2: This step covers browsing of nodes in the address space of a server using the Browse service.
•Step 3: Describes how to read and write simple variables.
•Step 4: Describes how to create Subscriptions and MonitoredItems to monitor attributes of nodes for changes (e.g. the value of a variable).
•Step 5: Shows how to handle custom types and their instances.
•Step 6: Shows how to call methods that are defined in the address space of an OPC UA server.
