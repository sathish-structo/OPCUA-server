Returns the maximum number of events returned for any node over the time range.

See also:

setMaxNumberOfValuesPerNode()

void SoftingOPCToolbox5::ReadEventDetails::setMaxNumberOfValuesPerNode(OTUInt32 maxNumberOfValuesPerNode)

Sets the maximum number of events returned for any node over the time range. If only one time is specified, the time range must extend to return this number of values. The default value of 0 indicates that there is no maximum.

Note:

At least two of the three parameters StartTime, EndTime and MaxNumberOfValuesPerNode must be specified.
