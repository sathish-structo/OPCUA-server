This event shall be generated if a client creates a session.

The SourceNode property for events of this type should be assigned to the Server object. The ClientUserId property for audit events is not available for this call thus this parameter shall be set to the "System/CreateSession".

Beside mandatory properties of the base class additionally the following mandatory properties shall be set:

•SecureChannelId: Shall uniquely identify the SecureChannel.
•ClientCertificate: The clientCertificate parameter of the CreateSession Service call.
•ClientCertificateThumbprint: A thumb print of the ClientCertificate.
•RevisedSessionTimeout: The returned revisedSessionTimeout parameter of the CreateSession Service call.
