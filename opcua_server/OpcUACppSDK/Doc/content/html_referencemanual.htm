This chapter documents the API that C++ programmers use to develop OPC UA-enabled applications. The reference manual contains the following sections:

•Data Type Overview: Gives an overview on data types have to be used when developing SDK applications
•Class Overview: Contains the complete list of API classes
•Global Functions: Describes the global functions of the API
•Common Classes: This section describes all classes that are used for both client and server development
•Client Classes: This section describes all classes that are used for client development
•Server Classes: This section describes all classes that are used for server development
•Data Type Classes: Describes all classes that represent OPC UA data types
•Enumerations: Documents all enumeration types and their values
