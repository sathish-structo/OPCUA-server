EventFilter Class Reference

Top  Previous  Next

#include "EventFilter.h"

 

The EventFilter data type is used by event MonitoredItems (see Client::MonitoredItem::setEventFilter()) to specify the content of reported events (select clauses) and criteria for an event to be reported (where clause).
