EnumUadpDataSetMessageContentMask Reference

Top  Previous  Next

defines the flags for the content of the DataSetMessage header.

EnumUadpDataSetMessageContentMask_Timestamp = 0x01

If this flag is set, a timestamp shall be included in the DataSetMessage header.

EnumUadpDataSetMessageContentMask_PicoSeconds = 0x02

If this flag is set, a PicoSeconds timestamp field shall be included in the DataSetMessage header.

This flag is ignored if the HeaderTimestamp flag is not set.

EnumUadpDataSetMessageContentMask_Status = 0x04

If this flag is set, the DataSetMessage status is included in the DataSetMessage header.

EnumUadpDataSetMessageContentMask_MajorVersion = 0x08

If this flag is set, the ConfigurationVersion.MajorVersion is included in the DataSetMessage header.

EnumUadpDataSetMessageContentMask_MinorVersion = 0x10

If this flag is set, the ConfigurationVersion.MinorVersion is included in the DataSetMessage header.

EnumUadpDataSetMessageContentMask_SequenceNumber = 0x20

If this flag is set, the DataSetMessageSequenceNumber is included in the DataSetMessage header.
