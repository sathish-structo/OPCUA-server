The OPC UA C++ Server & Client SDK V6.00 can be purchased including the source code in order to compile it with custom settings or for specific platforms.

While the Server source code license gives you full access to the OPC UA C++ Server & Client SDK V6.00 source code, the Client source code license does not deliver complete source code of server side components and it cannot be used for building Server applications.

The installation directory of the SDK must not contain special characters like blanks, "#", ...
The SDK itself works well in such installation paths. In case you want to compile OpenSSL it will fail since the OpenSSL makefiles do not support such special characters.
