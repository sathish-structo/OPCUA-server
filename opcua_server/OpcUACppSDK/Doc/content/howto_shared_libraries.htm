OPC UA C++ Server & Client SDK V6.00

Shared Libraries of SDK

Top  Previous  Next

For the Linux operating system, the OPC UA C++ Server & Client SDK V6.00 provides the SDK libraries also as shared libraries.

 

As the SDK uses the libraries of OpenSSL and libxml2, the shared libraries of the SDK require the shared libraries of OpenSSL and libxml2 of the correct versions to be installed.

The SDK uses the OpenSSL version 1.1.1o and the libxml2 version 2.9.11.
You should either retrieve the compiled packages of these libraries for your linux version or download and build the libraries from source code; you can download the sources from http://www.openssl.org/source and http://www.xmlsoft.org/
 

In case you have the Source Code License of the OPC UA C++ Server & Client SDK V6.00 and want to build the shared libraries of the SDK, see Compile the SDK from Source Code on Linux.
