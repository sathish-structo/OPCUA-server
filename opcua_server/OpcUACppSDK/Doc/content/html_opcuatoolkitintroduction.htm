Introduction to the SDK

Top  Previous  Next

This chapter gives an overview on OPC UA C++ Server & Client SDK. This introduction covers the following topics:

•Component Overview: This topic identifies the components that make up the SDK.
•Installation: Covers the installation procedures for the various operating systems and describes the installed files.
•Setting up a Project: Describes how to setup a project to use the SDK in your own application.
•Setting up a Certificate Store: Describes how to setup a certificate store for the OPC UA application instance certificates.
•Introduction to the C++ API: Gives a brief introduction to the C++ API and memory management concepts used in the SDK.
•Deployment: Documents the files that need to be distributed with an application that uses the SDK.
