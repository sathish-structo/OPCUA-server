EnumUadpNetworkMessageContentMask Reference

Top  Previous  Next

Defines the optional header fields to be included in the NetworkMessages produced by the WriterGroup.

EnumUadpNetworkMessageContentMask_PublisherId = 0x01

The PublisherId is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_GroupHeader = 0x02

The GroupHeader is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_WriterGroupId = 0x04

The WriterGroupId is included in the NetworkMessages. The flag is only valid if Bit 1 is set.

EnumUadpNetworkMessageContentMask_GroupVersion = 0x08

The GroupVersion is included in the NetworkMessages. The flag is only valid if Bit 1 is set.

EnumUadpNetworkMessageContentMask_NetworkMessageNumber = 0x10

The NetworkMessageNumber is included in the NetworkMessages.

The field is required if more than one NetworkMessage is needed to transfer all DataSets of the group. The flag is only valid if Bit 1 is set.

EnumUadpNetworkMessageContentMask_SequenceNumber = 0x20

The SequenceNumber is included in the NetworkMessages. The flag is only valid if Bit 1 is set.

EnumUadpNetworkMessageContentMask_PayloadHeader = 0x40

The PayloadHeader is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_Timestamp = 0x80

The Timestamp is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_PicoSeconds = 0x100

The PicoSeconds is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_DataSetClassId = 0x200

The DataSetClassId is included in the NetworkMessages.

EnumUadpNetworkMessageContentMask_PromotedFields = 0x400

The PromotedFields is included in the NetworkMessages.
