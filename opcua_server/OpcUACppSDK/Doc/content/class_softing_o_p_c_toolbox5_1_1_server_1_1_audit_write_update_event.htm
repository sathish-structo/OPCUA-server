#include "ServerAuditWriteUpdateEvent.h"

 

This event may be generated if a client changes the value of any attribute.

Beside properties of the base class additionally the following mandatory properties shall be set:

•AttributeId: The property that shall contain the attribute that was written on the SourceNode.
•IndexRange: The property that shall contain the index range of the written attribute.
•NewValue: The property that shall contain the value that was written to the SourceNode.
•OldValue: The property that shall contain the value that the SourceNode contained before the write.
