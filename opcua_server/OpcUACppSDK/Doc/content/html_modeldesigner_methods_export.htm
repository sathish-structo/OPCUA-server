Export

Top  Previous  Next

The Export method is used to export specific namespaces of the address space. The handleExportCustomNodes on the AddressSpaceRoot is called with a ExportCustomNodesTransaction. If the Server has an dynamic AddressSpace the application must export the Virtual Nodes by using the export methods in the transaction.

Input Parameters:

NamespaceUris

An array of Namespace uris that should be exported to the XML file

ExportFile

The (path and) filename of the XML file

GlobalExtensions

An array of XmlElements to be written in the global "Extension" tags.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

 

For description of "GlobalExtensions" see Import.

 

The OPC UA C++ Server & Client SDK V6.00 does not verify the content any given XmlElement. Malformed XmlElements will cause a malformed XML file.
