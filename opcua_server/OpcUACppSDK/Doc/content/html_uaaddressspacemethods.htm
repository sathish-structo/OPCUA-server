Methods

Methods represent functions that can be called by a client. They may have properties that describe their input and output arguments.

Besides the common attributes, methods expose the following attributes:

Attribute

Description

Executable

Indicates if the method is executable; it does take user rights into account. Typically, methods are executable, unless their execution is somehow disabled by the underlying hardware.

UserExecutable

Indicates if the current user has permissions to execute the method.
