virtual void SoftingOPCToolbox5::INetworkAddressUrlDataType::setUrl(const tstring & url) [pure virtual]

Sets the Url member in the class.

Parameters:

url

The Url defines both the protocol and target / source of the data. Typical URL are

•opc.udp://<host>[:<port>]
•amqps://<domain name>[:<port>][/<path>]
•mqtts://<domain name>[:<port>][/<path>
•wss://<domain name>[:<port>][/<path>]

Note:

For UDP using multi-cast or broadcast both publisher and subscriber shall configure same URL. For UDP uni-cast publisher shall configure host (IP address) of subscriber and vice versa.
