EnumStatusCode SoftingOPCToolbox5::Server::BrowseNextRequest::getContinuationPoint(BrowseContinuationPointPtr & continuationPoint) const

Returns the existing continuation point for the request.

Parameters:

continuationPoint (out)

The continuation point for the request

Returns:

A good result, if a continuation point exists. EnumStatusCode_BadContinuationPointInvalid, if the continuation point provided by the client has been invalid.

EnumStatusCode SoftingOPCToolbox5::Server::BrowseNextRequest::deleteContinuationPoint()

Deletes the continuation point for the request.

If the server application has returned all references to the client, it must delete the continuation point before completing the request.

Returns:

A good result, if the continuation point could be deleted. EnumStatusCode_BadContinuationPointInvalid, if the continuation point provided by the client has been invalid.
