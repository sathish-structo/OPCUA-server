UserNameIdentityToken Class Reference

Top  Previous  Next

#include "UserNameIdentityToken.h"

 

The UserNameIdentityToken data type is used to pass username/password credentials to the server.

UA defines various tokens for user identification. This one is using a user name and a password for identification.

The password is transported encrypted to avoid a "man in the middle" seeing the password.

virtual void SoftingOPCToolbox5::UserNameIdentityToken::setPolicyId(const tstring & value) [virtual]

Sets the identifier for the UserTokenPolicy that the token conforms to.

The PolicyId is an identifier, specified by the user identities returned from server during GetEndpoints service call (see Application::getEndpointsFromServer()). This value is only unique within the context of a single server.
