virtual void SoftingOPCToolbox5::ViewDescription::setViewVersion(OTUInt32 value) [virtual]

Sets the version number for the View desired.

When nodes are added to or removed from a View, the value of a View's ViewVersion property is updated. The current view is used if both Timestamp and ViewVersion is not set.

See also:

setTimestamp()

Note:

Either the Timestamp or the ViewVersion parameter may be set by a Client, but not both.
