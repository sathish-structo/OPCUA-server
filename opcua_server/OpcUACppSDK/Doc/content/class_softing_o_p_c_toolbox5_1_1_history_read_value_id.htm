#include "HistoryReadValueId.h"

 

The HistoryReadValueId data type is used by the HistoryRead services (see Client::Session::historyReadRaw(), Client::Session::historyReadProcessed(), and Client::Session::historyReadEvent()) to specify the node on which the historical retrieval is to be performed.
