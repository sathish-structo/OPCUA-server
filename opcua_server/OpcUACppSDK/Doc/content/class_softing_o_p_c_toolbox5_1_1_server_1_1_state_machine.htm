virtual EnumStatusCode SoftingOPCToolbox5::Server::StateMachine::deactivate() [virtual]

Deactivates the state machine. This method resets the values of the CurrentState and the LastTransition and assigns the values a bad quality (Bad_StateNotActive).

bool SoftingOPCToolbox5::Server::StateMachine::isActive() const

Returns whether the state machine is active.

virtual EnumStatusCode SoftingOPCToolbox5::Server::StateMachine::activate() [virtual]

Activates the state machine and sets the CurrentState property. Please note that this method does not touch the LastTransitionTime property.

This method returns a bad status code if the state machine is already active.
