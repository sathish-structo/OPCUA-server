Configure ReaderGroup

Top  Previous  Next

The ReaderGroup is used to group a list of DataSetReaders. It is not symmetric to a WriterGroup and is not related to a particular NetworkMessage as the filter settings are configured on the DataSetReaders.

Currently message signing and encrypting is not supported, security mode should be set to "No security" (EnumMessageSecurityMode_None).

The mandatory configuration requires to set the following fields:

•Define the maximum NetworkMessage size large enough to fit the binary data of the expected NetworkMessages (see Group::setMaxNetworkMessageSize()).
•Assign the DataSetReaders that shall be grouped for processing ReaderGroup::addDataSetReader() (see Configure DataSetReader).

 

PubSub::ReaderGroupPtr readerGroup = PubSub::ReaderGroup::create();

readerGroup->setName(_T("ReaderGroup_0"));

readerGroup->setEnabled(true);

readerGroup->setSecurityMode(EnumMessageSecurityMode_None);

readerGroup->setMaxNetworkMessageSize(500);

readerGroup->addDataSetReader(reader);
