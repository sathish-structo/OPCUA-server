EnumStatusCode SoftingOPCToolbox5::Server::TransitionVariable::reset()

Resets the components of this variable.

This method resets

•the Id property to a NULL value
•the Name property to an empty string
•the Number property to 0
•the TransitionTime property to 1601-01-01 12:00AM
