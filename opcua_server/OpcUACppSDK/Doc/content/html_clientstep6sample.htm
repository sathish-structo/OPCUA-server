This topic belongs to Calling a Method. It refers to the sample application Client_6 and should be tested with the sample application Server_4. (See Programming Tutorial for where to find the tutorial samples)

Where to Find

Source code and project files belonging to the sample application are located in the directory <InstallDir>/Apps/Samples/Cpp/Tutorial/Client_6.

Running the Client

•Compile and start the sample applications Client_6 and Server_4. Run the client application in the debugger and step through the code.
•Connect to the server with the command "co" and get the NodeIds of the variables, instances and components by calling a browse with "br".
•Use the command "op" (open) or "cl" (close) to execute the "OpenCloseDoor" method of a refrigerator instance. These commands have an optional parameter to specify the target refrigerator.

Points of Interest

•The relevant code is implemented in Refrigerator::openDoorMethod() in the file Refrigerator.cpp.
•How a single method call (Client::Session::callMethod()) is parametrized and called and how the results of the service call are consumed. (See Refrigerator::openDoorMethod())
