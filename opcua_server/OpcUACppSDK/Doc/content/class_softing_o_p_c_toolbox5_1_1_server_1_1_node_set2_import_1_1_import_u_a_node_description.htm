EnumNodeClass SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getNodeClass() const

Returns the class of the node. The possible node classes are defined by the Enumeration "EnumNodeClass". The set of node classes is fixed and cannot be extended. The returned information can be used to cast the common ImportUANodeDescription to a matching subtype.

const INodeId * SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getNodeId() const

Returns the NodeId of the node that uniquely identifies the node in the address space of the server. A NodeId is always defined within a namespace.

const IQualifiedName * SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getBrowseName() const

Returns the human-readable, non-localized browse name of the node. This name is used by clients to create paths out of browse names.

const ILocalizedText * SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getDisplayName() const

Returns the localized name of the node that shall be used in user interfaces. The DisplayName is localized and can contain more than one language.

EnumWriteMask SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getWriteMask() const

Returns the write mask of the node which indicates the attributes that can be written by a client.

The write mask is a bit field. A set bit in that field indicates the writability of a specific attribute (see EnumWriteMask).

Note:

Access rights on the 'value' attribute of a variable are defined by its access level attribute (see Variable::setAccessLevel()).

EnumWriteMask SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getUserWriteMask() const

Returns the user write mask of the node which indicates the attributes that can be written by a certain user.

See also:

getWriteMask()

const ILocalizedText * SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getDescription() const

Returns a description of the node that indicates the meaning of the node.

const std::vector< ImportReferenceDescription > & SoftingOPCToolbox5::Server::NodeSet2Import::ImportUANodeDescription::getReferences()

Returns the references, which are attached to this node.

Note:

Only the references are returned, which are described on this node. References from different nodes, which point to this node are not included.
