EnumIdentityMappingType Reference

Top  Previous  Next

Specifies the type criteria contained in the rule in IdentityMappingRuleType.

EnumIdentityMappingType_UserName = 1

The rule specifies a UserName from a UserNameIdentityToken.

EnumIdentityMappingType_Thumbprint = 2

The rule specifies the Thumbprint of a User or CA Certificate.

EnumIdentityMappingType_Role = 3

The rule is a Role specified in an Access Token (JSON Web Token). This type is not supported by SDK.

EnumIdentityMappingType_GroupId = 4

The rule is a user group specified in the Access Token (JSON Web Token). This type is not supported by SDK.

EnumIdentityMappingType_Anonymous = 5

The rule specifies Anonymous UserIdentityToken.

EnumIdentityMappingType_AuthenticatedUser = 6

The rules specify any non-Anonymous UserIdentityToken.
