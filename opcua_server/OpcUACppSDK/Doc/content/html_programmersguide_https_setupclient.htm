Setup Client Application

Top  Previous  Next

For the client application, there are two configuration steps required to be able to establish and use HTTPS connection.

1. Initialize the PKI Store

For HTTPS, the client application requires a PKI store to validate the HTTPS certificates of servers, see Application::setPkiStoreConfiguration().

The method must be set before initializing the application.

// during initialization

PkiStoreConfiguration storeConfiguration;

storeConfiguration.setCertificateTrustListLocation(_T("./PKI/trusted/"));

Application::instance()->setPkiStoreConfiguration(&storeConfiguration);

// ...

Application::instance()->initialize(&appDesc);

The PKI store must contain the HTTPS certificate of the server to allow the establishment of a connection.

2. Disable Channel Sharing

Disable the SecureChannel sharing between Session to reduce the number of requests per SecureChannel, see Application::setChannelSharing().

Application::instance()->setChannelSharing(false);

 

After these two steps, the client can connect to an HTTPS endpoint.
