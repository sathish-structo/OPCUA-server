EnumRequestState Reference

Top  Previous  Next

Enumeration for the state of a request (server only).

The initial state is "Initialized" and the final state "Completed". The two other states can be used by the user but are not evaluated internally.

EnumRequestState_Started = 1

The request has been started.

This is the state in which the request is when the callback on server side is performed.

EnumRequestState_Pending = 2

This flag indicates that a request (Read / Write Request or method call), is still in a pending state.

EnumRequestState_Completed = 3

The request has been completed.

If all requests of a transaction are in a completed state the Transaction's result is sent to the connected client.
