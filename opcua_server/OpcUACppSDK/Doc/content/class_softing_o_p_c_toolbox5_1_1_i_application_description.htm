IApplicationDescription Class Reference

#include "IApplicationDescription.h"

 

The ApplicationDescription data type is used to characterize an OPC UA application. The application type (see setApplicationType()) defines whether the application is a server and/or client application and is used during initialization (see Application::initialize()).
