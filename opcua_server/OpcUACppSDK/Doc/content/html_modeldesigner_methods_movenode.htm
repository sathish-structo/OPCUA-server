MoveNode

Top  Previous  Next

The MoveNode method is used to move a node to a different parent.
All references between old parent and the node to move are relocated to the new parent node.
Any other references are left untouched. Thus this method moves the complete tree to the new parent.

Input Parameters:

NodeToMove

The NodeId of the node that shall be moved.

SourceParentNodeId

The NodeId of the current parent node of the node to move.
This parameter is only required when the node to move has several parent nodes, otherwise a NULL value or a NULL NodeId can be set.

TargetParentNodeId

The NodeId of the new parent node, where the node shall be moved to.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.
