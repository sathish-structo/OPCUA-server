EnumStatusCode SoftingOPCToolbox5::Server::StateVariable::reset()

Resets the components of this variable.

This method resets

•the Id property to a NULL Value
•the Name property to an empty string
•the Number property to 0
•the EffectiveDisplayName property to an empty string
