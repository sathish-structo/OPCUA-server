#include "DataTypeDescription.h"

 

This is the base class of a user defined data type.

A server application can define and expose data types by attaching is to a DaatType node. Before assigning to a data type node, the data type cannot be used.

QualifiedName SoftingOPCToolbox5::DataTypeDescription::getName() const

Returns the Name of the DataTypeDescription

The name of the data type description is assigned at server side via Server::DataType::setDataTypeDescription(). Before this, an empty QualifiedName will be returned. On client side the name is provided by SDK with ClientSession::updateDataTypes().

NodeId SoftingOPCToolbox5::DataTypeDescription::getDataTypeId() const

Gets the NodeId of the data type node

Note:

For server application, the data type NodeId is available only after assigning the data type description to the data type node.
