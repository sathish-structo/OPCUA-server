#include "IViewDescription.h"

 

The ViewDescription data type is used by the Browse (see Client::Session::browse() et al.) and Query services (see Client::Session::queryFirst()) to specify the View to browse or query.

virtual void SoftingOPCToolbox5::IViewDescription::setViewVersion(OTUInt32 value) [pure virtual]

Sets the version number for the View desired.

When nodes are added to or removed from a View, the value of a View's ViewVersion property is updated. The current view is used if both Timestamp and ViewVersion is not set.

See also:

setTimestamp()

Note:

Either the Timestamp or the ViewVersion parameter may be set by a Client, but not both.
