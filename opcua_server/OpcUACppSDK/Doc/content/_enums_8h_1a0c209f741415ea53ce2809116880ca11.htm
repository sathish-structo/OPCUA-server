EnumNodeClass Reference

Top  Previous  Next

Enumeration for the different node classes (= nodes in the server).

Each Node has an attribute which identifies the node type by this enumeration type.

EnumNodeClass_Node = 0

Base node class. A concrete node can not be of that type.

EnumNodeClass_Object = 1

Object node class (see Object class definition)

EnumNodeClass_Variable = 2

Variable node class (see Variable class definition).

EnumNodeClass_Method = 4

Method node class (see Method class definition).

EnumNodeClass_ObjectType = 8

ObjectType node class (see ObjectType class definition).

EnumNodeClass_VariableType = 16

VariableType node class (see VariableType class definition).

EnumNodeClass_ReferenceType = 32

ReferenceType node class (see ReferenceType class definition).

EnumNodeClass_DataType = 64

DataType node class (see DataType class definition).

EnumNodeClass_View = 128

View node class (see View class definition).
