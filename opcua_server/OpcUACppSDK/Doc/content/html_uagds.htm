Global Discovery Server

Top  Previous  Next

A Global Discovery Server (GDS) maintains a list of OPC UA applications available in an administrative domain.

This list of applications is used for two main purposes:

Discovery: An OPC UA application can query the GDS to find other OPC UA applications of interest from the administrative domain.

Certificate Management: The Certificate Management is responsible to create, update and distribute certificates and trust lists

 

The full details of the GDS is described in the OPC UA Specification Part 12 - Discovery and Global Services.
