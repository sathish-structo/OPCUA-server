AddNamespace

Top  Previous  Next

The AddNamespace method is used to create a new namespace in the address space.

Input Parameters:

NamespaceUri

The URI of the new namespace to add

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

NamespaceIndex

The index of the newly created namespace, or if the namespace already existed before, the index of the existing one
