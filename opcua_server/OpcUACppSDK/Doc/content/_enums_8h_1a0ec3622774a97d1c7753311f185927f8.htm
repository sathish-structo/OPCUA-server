AccessLevelMask Reference

Top  Previous  Next

A set of bit masks to filter EnumAccessLevel.

AccessLevelMask_UserAccessLevel = EnumAccessLevel_CurrentRead | EnumAccessLevel_CurrentWrite | EnumAccessLevel_HistoryRead | EnumAccessLevel_HistoryWrite

A bit mask to filter the valid AccessLevel bits for the UserAccessLevel attribute of a Variable node.

AccessLevelMask_AccessLevel = AccessLevelMask_UserAccessLevel | EnumAccessLevel_SemanticChange

A bit mask to filter the valid AccessLevel bits for the AccessLevel attribute of a Variable node.

AccessLevelMask_UserExecutable = EnumAccessLevel_Executable

A bit mask to filter the valid AccessLevel bits for the UserExecutable attribute of a Method node.

AccessLevelMask_FullAccess = AccessLevelMask_AccessLevel | AccessLevelMask_UserExecutable

A bit mask to filter the valid AccessLevel bits of all non user specific attributes.

AccessLevelMask_FullUserAccess = AccessLevelMask_UserAccessLevel | AccessLevelMask_UserExecutable

A bit mask to filter the valid AccessLevel bits of all user attributes.
