IHistoryEventFieldList Class Reference

Top  Previous  Next

#include "IHistoryEventFieldList.h"

 

The HistoryEventFieldList data type contains a list of historical events being delivered to a client as result of a HistoryRead service call (see Client::Session::historyReadEvent()). The size and order of the elements in this list is determined by the select clauses in the EventFilter (see ReadEventDetails::setFilter()).
