The WriterGroupMessageDataType is an abstract base type without fields for message mapping specific WriterGroup parameters.

Please use concrete derived classes to configure specific parameters. Concrete subtypes are UadpWriterGroupMessageDataType and JsonWriterGroupMessageDataType (not yet supported)
