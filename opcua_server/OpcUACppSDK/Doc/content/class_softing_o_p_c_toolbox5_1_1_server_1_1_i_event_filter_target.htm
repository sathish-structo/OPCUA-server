IEventFilterTarget Class Reference

Top  Previous  Next

#include "IEventFilterTarget.h"

 

This interface shall be implemented by classes that provide information about an historical event. The interface is used by the EventFilterEvaluator to decide whether a historical event is included in a HistoryRead response or not.

virtual EnumStatusCode SoftingOPCToolbox5::Server::IEventFilterTarget::getAttributeValue(const std::vector< QualifiedName > & browsePath, EnumAttributeId attributeId, Value & attributeValue) [pure virtual]

Returns the attribute value of a node identified by the given browsePath. The browse path is given relative to the historical event. If the browse path cannot be resolved and/or information for the attributeId is not available in the historian, the value should be set to type StatusCode with the value EnumStatusCode_BadNoData.

Parameters:

browsePath

The browse path that identifies the node

attributeId

The attribute that the caller is interested in

attributeValue (out)

The requested attribute value or EnumStatusCode_BadNoData if the value is unavailable

Note:

Returning a non-good status code will cause the filter evaluation to fail.
