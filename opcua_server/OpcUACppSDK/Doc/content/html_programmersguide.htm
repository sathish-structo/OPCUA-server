This section introduces the additional features of the OPC UA C++ Server & Client SDK V6.00 which are not covered by the Programming Tutorial which describes only the features that are mostly used by applications.

This section includes following sub sections:

•Security in OPC UA Applications: Describes how to establish and use a secured connection and how to authenticate application and users on client and server side and how to authorize different users / user groups on a server.
•Alarms & Conditions: Describes the configuration and handling of alarms and conditions on server side and how to receive and handle them on client side.
•Model Import: Describes how to import an address space model to the servers address space.
•Programming Samples: Includes topics which are related to sample applications that show how to implement a certain feature. These topics describe how to run the applications and which points are of interest to look at.
•Structured Data Types: Includes the definition of structured Data Types and the access to custom structured Data Type Values.
•How To Use: Is a section with useful small articles describing various topics
•HTTPS Transport Protocol: Describes how to use the HTTPS protocol and the requirements, benefits and restrictions compared to opc.tcp.
•PubSub: Contains topics, for how to configure a publisher or subscriber for the PubSub communication model.
