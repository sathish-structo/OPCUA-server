virtual void SoftingOPCToolbox5::BrowseDescription::setIncludeSubtypes(bool value) [virtual]

Indicates whether subtypes of the specified reference type (see setReferenceTypeId()) are included in the list of returned references.

If set to true, all subtypes of the given reference type are included . If set to false, only the exact matching reference type is considered during the browse.

virtual void SoftingOPCToolbox5::BrowseDescription::setResultMask(EnumResultMask value) [virtual]

Sets a bit mask that indicates which kind of information about a node is returned in the browse call.

The following bits may be set in the bit mask:

•EnumResultMask_ReferenceType
•EnumResultMask_IsForward
•EnumResultMask_NodeClass
•EnumResultMask_BrowseName
•EnumResultMask_DisplayName
•EnumResultMask_TypeDefinition
•EnumResultMask_All
