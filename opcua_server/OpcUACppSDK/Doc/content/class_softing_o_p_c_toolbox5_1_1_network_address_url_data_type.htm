#include "NetworkAddressUrlDataType.h"

 

This class is a concrete subclass of NetworkAddressDataType.

It is used to store network address information in the form of an URL String.

The NetworkInterface member defines the network interface (IP address) which shall be used to send / receive data The URL member defines both the protocol and target / source of the data. Typical URL are

•opc.udp://<host>[:/<port>]
•amqps://<domain name>[:<port>][/<path>]
•mqtts://<domain name>[:<port>][/<path>
•wss://<domain name>[:<port>][/<path>]

virtual void SoftingOPCToolbox5::NetworkAddressUrlDataType::setUrl(const tstring & url) [virtual]

Sets the Url member in the class.

Parameters:

url

The Url defines both the protocol and target / source of the data. Typical URL are

•opc.udp://<host>[:<port>]
•amqps://<domain name>[:<port>][/<path>]
•mqtts://<domain name>[:<port>][/<path>
•wss://<domain name>[:<port>][/<path>]

Note:

For UDP using multi-cast or broadcast both publisher and subscriber shall configure same URL. For UDP uni-cast publisher shall configure host (IP address) of subscriber and vice versa.
