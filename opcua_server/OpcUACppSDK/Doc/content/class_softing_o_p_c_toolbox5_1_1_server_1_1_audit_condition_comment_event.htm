#include "ServerAuditConditionCommentEvent.h"

 

This event shall be generated if a comment was added to a condition.

Beside mandatory properties of the base class additionally the following mandatory properties shall be set:

•EventId: This property shall contain the id of the event for which the comment was added.
•Comment: Contains the actual comment that was added.
