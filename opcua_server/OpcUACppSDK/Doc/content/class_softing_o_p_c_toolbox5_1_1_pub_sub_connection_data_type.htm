virtual EnumStatusCode SoftingOPCToolbox5::PubSubConnectionDataType::setPublisherId(const IValue * publisherId) [virtual]

Sets the PublisherId member in the class.

The PublisherId is a unique identifier for a Publisher within a Message Oriented Middleware. It can be included in sent NetworkMessage for identification or filtering. The value of the PublisherId is typically shared between PubSubConnections but the assignment of the PublisherId is vendor specific.

Parameters:

publisherId

Publisher ID of this connection. Valid data types are OTUInt8, OTUInt16, OTUInt32, OTUInt64 and String

virtual void SoftingOPCToolbox5::PubSubConnectionDataType::setTransportProfileUri(const tstring & transportProfileUri) [virtual]

Sets the TransportProfileUri member in the class

Parameters:

transportProfileUri

The TransportProfileUri parameter indicates the transport protocol mapping and the message mapping used.
•http://opcfoundation.org/UA-Profile/Transport/pubsub-udp-uadp
•http://opcfoundation.org/UA-Profile/Transport/pubsub-mqtt-uadp
•http://opcfoundation.org/UA-Profile/Transport/pubsub-mqtt-json

Note:

Currently the SDK only supports the UDP-UADP transport profile.

The concrete NetworkAddressUrlDataType provides in the URL member also the protocol. Currently the transport profile URI is ignored by the SDK and only the protocol of the NetworkAddressUrlDataType URI is used.
