virtual EnumStatusCode SoftingOPCToolbox5::CallMethodRequest::addInputArgument(const IValue * pValue) [virtual]

Adds an input argument to the list of input argument values.

The size and order of this list matches the size and order of the input arguments defined by the InputArguments Property of the Method. The name, a description and the data type of each argument are defined by the Argument data type in each element of the method's InputArguments property.
