FilterOperand Class Reference

Top  Previous  Next

#include "FilterOperand.h"

 

The FilterOperand data type is used to specify a filter operand in a where clause of an event or node filter.

The FilterOperand can hold one of the following operand types:

•ILiteralOperand
•IElementOperand
•IAttributeOperand (not allowed for event filters)
•ISimpleAttributeOperand
