This topic belongs to Subscription to Data Changes. It refers to the sample application Client_4 and should be tested with the sample application Server_4. (See Programming Tutorial for where to find the tutorial samples)

Where to Find

Source code and project files belonging to the sample application are located in the directory <InstallDir>/Apps/Samples/Cpp/Tutorial/Client_4.

Running the Client

•Compile and start the sample applications Client_4 and Server_4. Run the client application in the debugger and step through the code.
•Connect to the server with the command "co" and get the NodeIds of the variables by calling a browse with "br".
•Use the command "mtv" to monitor all variables in the 'Objects' folder and "stv" to stop monitoring the variables.

Points of Interest

•The relevant code can be found in createSubscription(), monitorVariable() and stopMonitoringVariable() in the file Client_4.cpp and in SampleSubscription::onDataChange().
•How to create a subscription and monitored items to monitor value changes of variables. (See createSubscription(), SampleSubscription::onDataChange() and monitorVariable())
•How to stop monitoring a variable and delete monitored items and subscriptions from the server. (See stopMonitoringVariable())
