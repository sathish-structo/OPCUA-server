AddReference

Top  Previous  Next

The AddReference method is used to add additional reference between two nodes.

Notes:
Only one hierarchical reference is permitted between instance nodes that is equal to or derived from type "Organizes" or "HasChild" . In case AddReference is called with such a reference type the new reference will replace the existing one.
A reference type derived from "HasTypeDefinition" cannot be added.

Input Parameters:

ReferenceNodeId

The ReferenceType of the reference to create.

SourceNodeId

The NodeId where the reference originates from.

TargetNodeId

The NodeId where the reference points to.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.
