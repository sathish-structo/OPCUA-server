CreateVariableType

Top  Previous  Next

The CreateVariableType method is used to create a new variable type.

Input Parameters:

BaseType

The BaseType has to be an existing VariableType node.

NodeId

The NodeId attribute of the node.

BrowseName

The BrowseName attribute of the node.

DisplayName

The DisplayName attribute of the node.

Description

The Description attribute of the node.

DataType

The DataType attribute of the node.

ValueRank

The ValueRank attribute of the node.
See Server::VariableType::setValueRank() for the meaning of the values.

ArrayDimensions

The ArrayDimensions attribute of the node.
See Server::VariableType::setArrayDimensions() for the meaning of the values.

Value

The Value attribute of the node.

IsAbstract

The IsAbstract attribute of the node, describing whether this type can be instantiated.

The attributes represent the most common attributes required to describe a VariableType.

Define a value for the attributes that shall be initialized with a specific value, otherwise set a NULL value or empty value to let the SDK generate a default value.

Some rarely required attributes are not listed by this method and can only be set via the method ModifyAttributes.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

NamespaceIndex

The index of the new created namespace, or if the namespace already existed before, the index of the existing one

NodeId

A NodeId with generated GUID identifier in the namespace with the highest namespace index.

 

Default attribute values:

Attribute

Value

NodeId

A NodeId with generated GUID identifier in the namespace with the highest namespace index.

BrowseName

The namespace and the string representation of the NodeId attribute

DisplayName

The string representation of the NodeId

Description

Empty

DataType

The data type of the parent VariableType

ValueRank

ValueRank_Scalar

ArrayDimensions

Array dimensions matching to the value rank without length limitations

Value

The value of the parent VariableType

IsAbstract

false
