Secure Channel

Top  Previous  Next

The SecureChannel service set, illustrated in the figure below, defines services that allow an OPC UA client to establish a communication channel to ensure the confidentiality and integrity of messages exchanged with the OPC UA server. This service set is composed of the following services:

•OpenSecureChannel
•CloseSecureChannel

OPC UA clients in affect select parameters that define the type of secure connection which is to be made by selecting a particular Endpoint to connect to.
