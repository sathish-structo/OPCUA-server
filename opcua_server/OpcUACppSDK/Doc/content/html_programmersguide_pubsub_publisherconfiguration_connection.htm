Configure Connection

Top  Previous  Next

The Connection describes how to send NetworkMessages to the subscribers and groups a list of WriterGroups, whose NetworkMessages shall reach the same target(s).

The Connection can be used for both, publishers and subscribers.

To configure the Connection for a publisher, you need to set the following fields:

•Invoke Connection::setPublisherId() to assign an ID for this publisher.
This ID needs to be unique within the same network and can be used by subscribers to filter messages.
The PublisherId allows values of the type UInteger (OTUInt8, OTUInt16, OTUInt32 or OTUInt64) and String.
Note: If you have several Connections, which publish data to different networks, then you can use the same PublisherId for all Connections.
•Assign the network address via Connection::setNetworkAddress(), currently the only supported concrete data type to set is the NetworkAddressUrlDataType.
The NetworkAddressUrlDataType has two fields:
oAssign the URL via NetworkAddressUrlDataType::setUrl().
The URL defines both the protocol and the target.
Currently the only supported protocol is UADP and the syntax is opc.udp://<host>[:<port>], which allows hostnames and IP addresses. The default port is 4840.
It allows unicast and multicast.
oAssign the desired network interface (network card) via NetworkAddressUrlDataType::setNetworkInterface().
Provide the IP address of your desired network interface; in case of an empty string, the "default" interface will be used, which is typically a real interface (not the loop-back).
Note: Network interface names like "eth0" are not supported.
•Add the WriterGroups, that shall use this Connection to publish notifications via Connection::addWriterGroup() (see Configure WriterGroup).

After the Connection is configured, you have to register this Connection to the Application via Application::addPubSubConnection().

Once everything is configured, you can start to publish by enabling the Connection via Connection::setEnabled() (see PubSubState)

PubSub::ConnectionPtr connection = PubSub::Connection::create();

connection->setPublisherId(Value((OTUInt32)13));

NetworkAddressUrlDataType address;

address.setUrl(_T("opc.udp://239.0.0.1");

connection.setNetworkAddress(address);

connection->addWriterGroup(writerGroup);

 

Application::instance()->addPubSubConnection(connection);

...

connection->setEnabled(true);
