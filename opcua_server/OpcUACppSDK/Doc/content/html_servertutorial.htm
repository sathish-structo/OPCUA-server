Server Tutorial

The server tutorial uses the Client Tutorial Step 6 Sample Application, to access the features implemented by the server tutorial. This tutorial shows step by step how to implement an OPC UA application, creating a customized address space (see Design of the Information Model), providing it with several features to be accessible for OPC UA clients and how to handle these features individually e.g. to connect them to a device.

 

Overview of the server tutorial steps

•Step 1: Initialize and terminate an OPC UA application and creating a connectable endpoint without security configuration.
•Step 2: Shows how to create simple nodes and also model based objects of a custom type.
•Step 3: How to handle read and write requests on Nodes to connect them to a device.
•Step 4: Creating an OPC UA method to perform a user defined action.
