virtual std::vector< EnumStatusCode > SoftingOPCToolbox5::ParsingResult::getDataStatusCodes() const [virtual]

Returns a list of results for the "DataToReturn" parameter of the NodeTypeDescription provided with the QueryFirst service call.

The size and order of the list matches the size and order of the QueryDataDescription elements provided with the "DataToReturn" parameter. The vector can be empty if no errors where encountered.

Reimplemented from SoftingOPCToolbox5::IParsingResult.

virtual std::vector< DiagnosticInfo > SoftingOPCToolbox5::ParsingResult::getDataDiagnosticInfos() const [virtual]

Returns a list of diagnostic information for the "DataToReturn" parameter of the NodeTypeDescription provided with the QueryFirst service call.

The size and order of the list matches the size and order of the QueryDataDescription elements provided with the "DataToReturn" parameter. The vector can be empty if no diagnostic information was encountered in processing the request.
