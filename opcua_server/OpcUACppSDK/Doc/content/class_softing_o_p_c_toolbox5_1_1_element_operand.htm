ElementOperand Class Reference

Top  Previous  Next

#include "ElementOperand.h"

 

The ElementOperand data type is used in the context of event or node filters (in queries) to link sub-elements within a ContentFilter.

The link is in the form of an integer that is used to index into the array of elements contained in the ContentFilter. An index is considered valid if its value is greater than the element index it is part of and it does not reference a non-existent element. Clients shall construct filters in this way to avoid circular and invalid References.
