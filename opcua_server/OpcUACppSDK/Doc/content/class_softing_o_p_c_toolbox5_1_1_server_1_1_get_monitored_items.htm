#include "ServerGetMonitoredItems.h"

 

This class represents the 'GetMonitoredItems' method to retrieve all ClientHandles and ServerHandles from all MonitoredItems of a Subscription.

This method is required to identify MonitoredItems, where the client does not know, whether the item was created or not and to retrieve the ServerHandle when only the ClientHandle is known. Such cases can occur when a CreateMonitoredItemsRequest times out and the client can not know if the server had processed the request or not. Publish responses for a Subscription contain only the ClientHandles for the MonitoredItems, if the client identifies an unknown MonitoredItem, it has to retrieve the corresponding ServerHandle to be able to delete it.
