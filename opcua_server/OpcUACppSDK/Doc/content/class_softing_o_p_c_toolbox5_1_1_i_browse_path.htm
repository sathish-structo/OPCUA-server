IBrowsePath Class Reference

Top  Previous  Next

#include "IBrowsePath.h"

 

The BrowsePath data type is used by the TranslateBrowsePathsToNodeIds service (see Client::Session::translateBrowsePathToNodeIds()) to specify a relative path (using browse names) from one node to another.
