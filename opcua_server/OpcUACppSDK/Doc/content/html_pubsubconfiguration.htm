Configuration

Top  Previous  Next

PubSub Configuration is an important topic when developing a Publisher or a Subscriber. The specification defines different configuration parameters that define the behavior of Publisher and Subscriber.

Configuration of these parameters can be performed through the OPC UA Information Model for PubSub configuration or through vendor-specific mechanisms.

The configuration parameters are grouped by component and define the data acquisition for the DataSets, the message generation and the sending on the Publisher side.
