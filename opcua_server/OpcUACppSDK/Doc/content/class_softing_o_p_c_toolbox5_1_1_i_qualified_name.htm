#include "IQualifiedName.h"

 

The QualifiedName data type describes a name that is qualified by a namespace. The namespace is identified by an index into the server's NamespaceArray. A QualifiedName is, for example, used as BrowseName of a node.

The NamespaceArray is a list of namespace URIs. Each URI identifies the naming authority responsible for assigning the identifier element of the NodeId. A server's NamespaceArray always contains the URI http://opcfoundation.org/UA/ at index 0 which is the OPC UA namespace that contains well-known NodeIds.

virtual EnumStatusCode SoftingOPCToolbox5::IQualifiedName::parse(const tstring & nameString) [pure virtual]

Parses the given string for a namespace index and name and sets the content to this QualifiedName instance.

The string syntax is [<NamespaceIndex>:]<Name>.

•<NamespaceIndex> is a decimal number from 0 to 65535. The <NamespaceIndex> and colon can be left out to use index 0.
•<Name> can contain any character except the reserved characters '/', '.', '<', '>', ':', '#', '!' and '&'. Any reserved character within <Name> needs to be escaped with the '&'. For example "Car.Wheel" needs to be escaped as "Car&.Wheel".

Examples: "5:MyName", "Objects", "3:Engine&:Temperature".

Note:

Only the reserved characters are allowed to be escaped.
