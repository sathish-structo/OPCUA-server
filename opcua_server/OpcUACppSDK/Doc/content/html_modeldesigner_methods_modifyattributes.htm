ModifyAttributes

Top  Previous  Next

The ModifyAttributes method can modify any attribute of any node type.

Input Parameters:

Node

The NodeId attribute of the node that shall be modified.

NodeId

The new NodeId attribute to assign to the node.

BrowseName

The BrowseName attribute of the node.

DisplayName

The DisplayName attribute of the node.

Description

The Description attribute of the node.

WriteMask

The WriteMask attribute of the node.
This bit mask stores a combination of EnumWriteMask values.

DataType

The DataType attribute of a Variable or VariableType.

ValueRank

The ValueRank attribute of a Variable or VariableType.
See Server::Variable::setValueRank() for the meaning of the values.

ArrayDimensions

The ArrayDimensions attribute of a Variable or VariableType.
See Server::Variable::setArrayDimensions() for the meaning of the values.

Value

The Value attribute of a Variable or VariableType.

AccessLevel

The AccessLevel attribute of a Variable.
This bit mask stores a combination of EnumAccessLevel values.

MinimumSamplingInterval

The MinimumSamplingInterval attribute of a Variable.

Historizing

The Historizing attribute of a Variable.

EventNotifier

The EventNotifier attribute of an Object.
This bit mask stores a combination of EnumEventNotifier values.

Executable

The Executable attribute of a Method.

IsAbstract

The IsAbstract attribute of the node, describing whether this type can be instantiated.

Symmetric

The Symmetric attribute, describing whether the references have the same meaning in both directions.

InverseName

The InverseName attribute of the ReferenceType.
This attribute is only used for ReferenceTypes where the Symmetric attribute is set to false.

ContainsNoLoops

The ContainsNoLoops attribute of a View.

DataTypeDefinition

The option DataTypeDefinition attribute of DataType to assign a specific StructureDescription or EnumDescription.

Define a new value for any attribute that shall be changed, otherwise set a NULL value or an empty value to keep the current attribute as it is.

Only the attributes of the targeted node type can be modified.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

NamespaceIndex

The index of the new created namespace, or if the namespace already existed before, the index of the existing one
