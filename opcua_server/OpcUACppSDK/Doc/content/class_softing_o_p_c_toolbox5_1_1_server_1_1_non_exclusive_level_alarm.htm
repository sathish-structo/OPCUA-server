A level alarm is commonly used to report when a limit is exceeded. It typically relates to an instrument (e.g. a temperature meter). The level alarm becomes active when the observed value is above a high limit or below a low limit.

The NonExclusiveLevelAlarm is a special level alarm utilized with one or more non-exclusive states. If for example both the High and HighHigh states need to be maintained as active at the same time this alarm type should be used.
