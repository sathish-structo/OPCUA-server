Configure WriterGroup

Top  Previous  Next

The WriterGroup groups a list of DataSetWriters and describes how to put the DataSetMessage of the DataSetWriters into one NetworkMessage (e.g. the UDP packet on the wire) and when to send the notifications.

The mandatory configuration requires to set the following fields:

•Set an ID via WriterGroup::setWriterGroupId(), the ID shall be unique across all WriterGroups for the same PublisherId (see Configure Connection). Subscribers can use this ID to filter messages.
•Set the publishing interval in milliseconds via WriterGroup::setPublishingInterval().
•Set the message settings via WriterGroup::setMessageSettings(), which define the protocol specific settings of the WriterGroup. Currently only the UadpWriterGroupMessageDataType is supported.
On the UadpWriterGroupMessageDataType, you should define the content of the NetworkMessage to send via UadpWriterGroupMessageDataType::setNetworkMessageContentMask().
Note: The default value 0 can theoretically be used, but we propose to set this mask more reasonable, see EnumUadpNetworkMessageContentMask to see all available flags:
oWe propose to always set the bits for the PublisherId, GroupHeader, WriterGroupId and PayloadHeader, except when there is only one subscriber for only one DataSet.
These bits mean to transport all IDs, so that subscribers are able to filter the contents.
oSelect the bit for the GroupVersion when the optional setter UadpWriterGroupMessageDataType::setGroupVersion() was invoked.
oSelect the bit for the DataSetClassId when the optional setter PublishedDataSet::setDataSetClassId() is used on any of the PublishedDataSets (at the DataSetWriters) of this WriterGroup.
•Assign the DataSetWriters that shall be published by this group via WriterGroup::addDataSetWriter() (see Configure DataSetWriter).

PubSub::WriterGroupPtr writerGroup = PubSub::WriterGroup::create();

writerGroup->setWriterGroupId(1);

writerGroup->setPublishingInterval(1000.0);

UadpWriterGroupMessageDataType defaultUadpGroupMessage;

uadpGroupMessageSettings.setNetworkMessageContentMask(EnumUadpNetworkMessageContentMask_PublisherId |

    EnumUadpNetworkMessageContentMask_GroupHeader |

    EnumUadpNetworkMessageContentMask_WriterGroupId |

    EnumUadpNetworkMessageContentMask_PayloadHeader);

writerGroup->setMessageSettings(uadpGroupMessageSettings);
