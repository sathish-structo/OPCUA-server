IDecimal Class Reference

Top  Previous  Next

#include "IDecimal.h"

 

The Decimal data type describes a high-precision signed decimal number.

It consists of an arbitrary precision integer unscaled value and an integer scale. The scale is the inverse power of ten that is applied to the unscaled value.

I.e. the decimal number is "value * 10^(-scale)".

virtual OTInt32 SoftingOPCToolbox5::IDecimal::getValueSize() const [pure virtual]

Get the size (in bytes) of the decimal's value.

Note:

When the decimal is initialized by a setter from an unsigned source value, getValueSize() will return 1 byte more than the base data type's size. Since 2's complement requires 1 bit more to store full range of unsigned types, the setter will use this extra byte when a change of bit length is allowed.
