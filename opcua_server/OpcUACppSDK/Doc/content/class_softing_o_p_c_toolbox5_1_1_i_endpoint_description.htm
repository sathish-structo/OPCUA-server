#include "IEndpointDescription.h"

 

The EndpointDescription data type describes the endpoint for a server and is returned by the GetEndpoints service (see Application::getEndpointsFromServer()).

An EndpointDescription contains all information a client needs to establish a session with the server.

The following constants are defined as convenience for the security policy URIs defined by the OPC UA specifications:
