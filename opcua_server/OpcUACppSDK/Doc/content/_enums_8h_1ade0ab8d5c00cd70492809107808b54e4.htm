EnumTraceGroup Reference

Top  Previous  Next

Defines the trace groups available in the SDK.

EnumTraceGroup_None = 0x00

This trace group reports no traces.

EnumTraceGroup_Core = 0x01

This trace group reports traces from the SDK's core.

EnumTraceGroup_Api = 0x02

This trace group reports traces from the SDK's C++ API.

EnumTraceGroup_Stack = 0x04

This trace group reports traces from the underlying OPC UA communication stack.

EnumTraceGroup_DiagnosticInformation = 0x08

This trace group is currently unused.

EnumTraceGroup_Statistics = 0x10

This trace group is currently unused.

EnumTraceGroup_User1 = 0x100

This trace group may be used by applications for their own purposes.

EnumTraceGroup_User2 = 0x200

This trace group may be used by applications for their own purposes.

EnumTraceGroup_User3 = 0x400

This trace group may be used by applications for their own purposes.

EnumTraceGroup_User4 = 0x800

This trace group may be used by applications for their own purposes.

EnumTraceGroup_ModelDesigner = 0x1000

This trace group reports traces from node set import, node set export and address space manipulation by the model designer.

EnumTraceGroup_PubSub_Publisher = 0x2000

This trace group reports traces from the PubSub Publisher.

EnumTraceGroup_PubSub_Subscriber = 0x4000

This trace group reports traces from the PubSub Subscriber.

EnumTraceGroup_All = 0x7FFF

This bit mask can be used to select all available trace groups.
