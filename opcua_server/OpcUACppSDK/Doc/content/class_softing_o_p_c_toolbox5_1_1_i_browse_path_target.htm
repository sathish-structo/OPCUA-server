#include "IBrowsePathTarget.h"

 

The BrowsePathTarget data type is used by the TranslateBrowsePathsToNodeIds service (see Client::Session::translateBrowsePathToNodeIds()) to hold the target node that matches a given browse path.

virtual OTUInt32 SoftingOPCToolbox5::IBrowsePathTarget::getRemainingPathIndex() const [pure virtual]

Returns the index of the first unprocessed element in the RelativePath. This value is equal to the maximum value of OTUInt32 if all elements were processed. Any other value indicates that the remaining path has to be processed on a remote server. The ServerIndex of the ExpandedNodeId identifies the server for further processing.
