#include "PubSubWriterGroup.h"

 

The WriterGroup class represents a PubSub WriterGroup for publishing values.

A writer group contains settings for a group of PubSub::DataSetWriter. The writer group is responsible to publish cyclically messages containing all data of the contained DataSetWriter objects.
