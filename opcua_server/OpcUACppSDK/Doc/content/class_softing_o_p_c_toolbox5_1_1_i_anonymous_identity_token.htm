IAnonymousIdentityToken Class Reference

#include "IAnonymousIdentityToken.h"

 

The AnonymousIdentityToken is used to indicate that the client has no user credentials. OPC UA defines various tokens for user identification. This one is using no identification.

virtual void SoftingOPCToolbox5::IAnonymousIdentityToken::setPolicyId(const tstring & value) [pure virtual]

Sets the policy identifier for this user identity token.

The policyId is an identifier, specified by the user identities returned from server during getEndpoints() call. This value is only unique within the context of a single server.
