This event shall be generated if a condition is shelved or unshelved.

Beside properties of the base class additionally the following optional property can be set:

•ShelvingTime: This property shall contain duration for which the Alarm is to be shelved.

EnumStatusCode SoftingOPCToolbox5::Server::AuditConditionShelvingEvent::setShelvingTime(double shelvingTime)

Sets the ShelvingTime property of the audit event.

The shelving time property is an optional property for this event type. In case of a TimeShelve operation the property shall contain the duration for which the alarm is to be shelved. For other changes in the shelving state this property can be omitted.
