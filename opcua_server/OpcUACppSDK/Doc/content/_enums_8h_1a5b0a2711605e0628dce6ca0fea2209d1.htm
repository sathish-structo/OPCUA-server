EnumDiagnosticsToReturn Reference

Top  Previous  Next

Defines bit masks that are used to request diagnostic information for service invocations.

EnumDiagnosticsToReturn_ServiceLevel_SymbolicId = 0x001

Return a namespace-qualified, symbolic identifier for an error or condition.

The maximum length of this identifier is 32 characters. The diagnostic info shall be related to a transaction (also known as service).

EnumDiagnosticsToReturn_ServiceLevel_LocalizedText = 0x002

Return up to 256 bytes of localized text that describes the symbolic id.

The diagnostic info shall be related to a transaction (also known as service).

EnumDiagnosticsToReturn_ServiceLevel_AdditionalInfo = 0x004

Return a byte string that contains additional diagnostic information, such as a memory image.

The format of this byte string is vendor-specific, and may depend on the type of error or condition encountered. The diagnostic info shall be related to a transaction (also known as service).

EnumDiagnosticsToReturn_ServiceLevel_InnerStatusCode = 0x008

Return the inner StatusCode associated with the transaction (also known as service).

EnumDiagnosticsToReturn_ServiceLevel_InnerDiagnostics = 0x010

Return the inner diagnostic info associated with the transaction (also known as service).

The contents of the inner diagnostic info structure are determined by other bits in the mask. Note that setting this bit could cause multiple levels of nested diagnostic info structures to be returned.

EnumDiagnosticsToReturn_OperationLevel_SymbolicId = 0x020

Return a namespace-qualified, symbolic identifier for an error or condition.

The maximum length of this identifier is 32 characters. The diagnostic info shall be related to a request (also known as operation).

EnumDiagnosticsToReturn_OperationLevel_LocalizedText = 0x040

Return up to 256 bytes of localized text that describes the symbolic id.

The diagnostic info shall be related to a request (also known as operation).

EnumDiagnosticsToReturn_OperationLevel_AdditionalInfo = 0x080

Return a byte string that contains additional diagnostic information, such as a memory image.

The format of this byte string is vendor-specific, and may depend on the type of error or condition encountered. The diagnostic info shall be related to a request (also known as operation).

EnumDiagnosticsToReturn_OperationLevel_InnerStatusCode = 0x100

Return the inner StatusCode associated with the request (also known as operation).

EnumDiagnosticsToReturn_OperationLevel_InnerDiagnostics = 0x200

Return the inner diagnostic info associated with the request (also known as operation).

The contents of the inner diagnostic info structure are determined by other bits in the mask. Note that setting this bit could cause multiple levels of nested diagnostic info structures to be returned.
