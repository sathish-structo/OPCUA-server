#include "IGuid.h"

 

The Guid data type defines a value that is a 128-bit Globally Unique Identifier. The static method Guid::generateGuid() may be used to generate a new GUID.

virtual EnumStatusCode SoftingOPCToolbox5::IGuid::parse(tstring guidString) [pure virtual]

Parses the given string and sets the parsed GUID on the Guid instance.

The string must conform to the following syntax: {hhhhhhhh-hhhh-hhhh-hhhh-hhhhhhhhhhhh} where h represents a hexadecimal figure between 0 and F. Example: {00000000-0000-0000-0000-000000000000}
