EnumTimeoutGroup_Publish = 0x80

Timeout for publish requests.

Publish requests are sent SDK internally to the server. Responses are notified to the client using Client::Subscription::onDataChange and Client::Subscription::onEventsReceived

Since publish requests are enqueued at server until new notifications are sent it is recommended to set a long timeout. Especially do not set this timeout smaller than the maximal used publish interval of a subscription.

Default: 60000 ms
