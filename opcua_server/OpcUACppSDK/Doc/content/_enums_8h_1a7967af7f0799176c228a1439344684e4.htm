EnumAccessLevel Reference

Top  Previous  Next

Bit coded enumeration (value can be a combination) defining the allowed access level to a value of a Variable or executable bit of a Method node.

EnumAccessLevel_None = 0x00

Empty value for init or clear of values.

EnumAccessLevel_CurrentRead = 0x01

Allows the read of the current value.

EnumAccessLevel_CurrentWrite = 0x02

Allows the write of the current value.

EnumAccessLevel_HistoryRead = 0x04

Allows a history read on the Variable.

EnumAccessLevel_HistoryWrite = 0x08

Allows a history write on the Variable.

EnumAccessLevel_SemanticChange = 0x10

Indicates if the Variable used as Property generates SemanticChangeEvents.

This bit is valid only for defining the AccessLevel but invalid for the UserAccessLevel

EnumAccessLevel_Executable = 0x80

Allows to execute the current method.

This bit is valid only for defining the UserAccessLevel
