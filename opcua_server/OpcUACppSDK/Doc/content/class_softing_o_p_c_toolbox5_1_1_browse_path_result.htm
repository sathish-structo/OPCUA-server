virtual std::vector< BrowsePathTarget > SoftingOPCToolbox5::BrowsePathResult::getTargets() const [virtual]

Returns the list of target nodes for the relative path from the starting node.

A server may encounter a reference to a node in another server which it cannot follow while it is processing the relative path. If this happens the server returns the NodeId of the external node and sets the RemainingPathIndex (see IBrowsePathTarget::getRemainingPathIndex()) parameter to indicate which relative path elements still need to be processed. To complete the operation the client shall connect to the other server and call this service again using the target as the starting node and the unprocessed elements as the relative path.
