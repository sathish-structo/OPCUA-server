#include "ServerMultiStateItem.h"

 

This class represents a DiscreteItem that can have more than two states. Each state is described by a LocalizedText.

Properties of MultiStateItem:

•EnumStrings (mandatory): A string lookup table corresponding to sequential numeric values (0, 1, 2, etc.). Example: {"OPEN", "CLOSE", "IN TRANSIT"}. Here the string "OPEN" corresponds to 0, "Close" to 1 and "IN TRANSIT" to 2.
