Configuring a Trace Endpoint

The Trace Viewer needs to be configured to expose a TCP endpoint that your application can connect to. Please select Add configuration from the file menu to set up an endpoint configuration:

•Name: Give the configuration a name. Use only characters that are allowed in filenames.
•Remote address and port number: Please provide an IP address and a port number for the trace endpoint. Please note that resolution of host names is currently not supported.
•Stored traceruns: This parameter limits the number of files which are stored on the file system. The oldest file is deleted if this number is reached.
•Start activated: When this option is selected, the trace endpoint is opened whenever the trace viewer application is started.
