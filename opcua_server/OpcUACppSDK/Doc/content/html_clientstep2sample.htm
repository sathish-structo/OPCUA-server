This topic belongs to Browsing the Address Space. It refers to the sample application Client_2 and should be tested with the sample application Server_4. (See Programming Tutorial for where to find the tutorial samples)

Where to Find

Source code and project files belonging to the sample application are located in the directory <InstallDir>/Apps/Samples/Cpp/Tutorial/Client_2.

Running the Client

•Compile and start the sample applications Client_2 and Server_4. Run the client application in the debugger and step through the code.
•Connect to the server with the command "co".
•Use the command "br" to execute the browse call.

Points of Interest

•How server nodes are identified (the concept of the NodeId, see Data Types).
•How to browse the address space of a server and how to consume the browse results. (See browse() in the file Client_2.cpp)
•How to handle continuation points is demonstrated in browseRefrigerators() in the file Client_5.cpp. (See Client Tutorial Step 5)
