This topic belongs to Custom Handling of Read Requests and Custom Handling of Write Requests. It refers to the sample application Server_2 and should be tested with the sample application Client_6. (See Programming Tutorial for where to find the tutorial samples)

Where to Find

Source code and project files belonging to the sample application are located in the directory <InstallDir>/Apps/Samples/Cpp/Tutorial/Server_2.

Running the Server

•Compile and start the sample applications Server_2 and Client_6. Run the server application in the debugger and step through the code.
•After connecting the client with the command "co" use the browse command "br" to display the created nodes (The order of the displayed Nodes is a bit separated due to the example handling of the browse call). After browsing the address space read the value of the variables with the command "rdv" and write a value with "wrv". The write command also has an optional parameter to specify the desired value to write.
•Use the commands "add" and "rem" in the server console to add variables to the address space and remove them during runtime. The "add" command has an additional parameter to specify the Name of the variable.

Points of Interest

•The related steps are implemented in initAddressSpace(), runUserInterface() and addVariable() in the file Server_2.cpp, MyVariable::handleReadRequest() and MyVariable::handleWriteRequest()
•How Server Nodes are identified (the concept of the NodeId and index of the UA namespaces, see Data Types).
•How a Node is added to the server's AddressSpace and how References between Nodes are set.
•Adding and removing nodes (here: variables) during runtime. (See runUserInterface(), initAddressSpace() and addVariable() in the file Server_2.cpp)
•The attributes that have to be populated for a server node. (See initAddressspace() and addVariable() in the file Server_2.cpp)
•The Variable class on the server side (nodes exposing values).
•How to handle a ReadRequest by overriding the handleReadRequest() method of the Variable class. (See handleReadRequest() in the file MyVariable.cpp)
•How to handle a WriteRequest by overriding the handleWriteRequest() method of the Variable class. (See handleWriteRequest() in the file MyVariable.cpp)
