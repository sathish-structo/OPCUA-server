EnumStatusCode SoftingOPCToolbox5::StructuredDataTypeDescription::setFields(const std::vector< StructureField > & fields)

Defines the fields of the structured data type or union

Parameters:

fields

The fields of the StructuredDataTypeDescription

Note:

The list of fields contain only the fields defined by this data type. In case of a derived structure, the fields of the base type shall not be specified in the description, but have to be provided in the Value.

If this StructuredDataTypeDescription is configured to describe a union, then the "IsOptional" flag of the fields is ignored.

All fields must have different names, including the fields of all base types.

EnumStatusCode SoftingOPCToolbox5::StructuredDataTypeDescription::getDefinition(StructureDefinition & definition) const

Gets the definition of the structured data type.

A structured data type is completely described by its definition. Beside the StructureFields, the definition also contains

•The default encoding ID, which is used for encoding / decoding values
•The base type of the structure, where the structure inherits fields from
•The StructureType, which gives more information about the structure

All these information cannot be set directly, but are automatically calculated once a server exposes a data type.

•The StructureType depends on existence of optional fields.
•The base type depends on the hierarchy of data type node, where the definition is assigned to.
•The encoding ID is provided in Server::DataType::setDataTypeDescription

Note:

The member DefaultEncodingId of definition shall match the NodeId provided when the description is assigned to a data type node via Server::DataType::setDataTypeDescription().

The member BaseDataType of definition is currently ignored. The base data type is calculated with the HasSubtype reference to the Server::DataType node.

NodeId SoftingOPCToolbox5::StructuredDataTypeDescription::getEncodingId() const

Gets the NodeId of the encoding node

Note:

For server application, the encoding NodeId is available only after assigning the data type description to the data type node.

See also:

Server::DataType::setDataTypeDescription()

Note:

The encoding NodeId may be the Null NodeId, especially for enumeration data types, where the encoding is implicitly done as Int32.

EnumStatusCode SoftingOPCToolbox5::StructuredDataTypeDescription::setUnion(bool descriptionIsUnion)

Defines if this StructuredDataTypeDescription describes a union (true) or a structure (false).

The default value is "false" to describe a structure.

Note:

Unions must be direct subtypes of data type "Union", so unions cannot be subtypes of other unions.
