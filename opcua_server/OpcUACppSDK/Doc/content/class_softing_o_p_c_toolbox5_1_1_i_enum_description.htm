virtual void SoftingOPCToolbox5::IEnumDescription::setBuiltInType(const OTUInt8 & builtInType) [pure virtual]

Sets the BuiltInType member in the class Parameters:

builtInType

The builtInType indicates if the DataType is an Enumeration or an OptionSet (not yet supported). If the builtInType is Int32, the DataType is an Enumeration. If the builtInType is one of the UInteger DataTypes or ExtensionObject, the DataType is an OptionSet.
