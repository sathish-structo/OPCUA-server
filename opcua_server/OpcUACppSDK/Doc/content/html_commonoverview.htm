OPC UA SDK Components

•TB5CPP: This component is delivered as binary only. It exposes the C++ API that is used to develop both OPC UA client and server applications. The C++ API provides functions for servers that wish to import and/or export OPC UA information models from XML files.
•TB5OT: This component is delivered as binary only. It is the core component of the SDK and implements the abstractions that are exposed by the SDK's C++ API. The component itself exposes a C API which is not intended to be used by customers.
•TB5Stack: This component is delivered as binary only. It implements the OPC UA transport protocols.
•TB5Util: This component is delivered as binary only. It provides a collection of utility classes and functions that are independent of the operating system.
•OpenSSL: This component is delivered as binary only. The OpenSSL library (Version 1.1.1o) is provided by the OpenSSL Project and implements security-related algorithms required by the OPC UA stack.
•libxml2: This component is delivered as binary only. The libxml2 (Version 2.9.11) is provided by the Libxml2 project and implements an XML parser.
