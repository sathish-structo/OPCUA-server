#include "ServerEventFilterEvaluator.h"

 

This class provides means to evaluate whether a (historical) event passes an event filter provided by the client or not. It is meant to be used by server applications in the context of a HistoryReadTransaction.

static EnumStatusCode SoftingOPCToolbox5::Server::EventFilterEvaluator::evaluate(const ContentFilter & eventFilter, IEventFilterTarget & historicalEvent, bool & eventPassesFilter)

Evaluates whether a (historical) event passes an event filter.

Parameters:

eventFilter

The event filter under which the historical event is to be evaluated

historicalEvent

The historical event that is to be evaluated

eventPassesFilter (out)

If true, the historical event passes the event filter.

Note:

A bad status code indicates that an error occurred during filter evaluation. In this case, the value of eventPassesFilter is undefined!
