AnalogItem Class Reference

Top  Previous  Next

#include "ServerAnalogItem.h"

 

This class represents a variable containing an analog value.

The AnalogItem variable has the following properties:

•InstrumentRange (optional): Defines the value range that can be returned by the instrument.
•EURange (mandatory): Defines the value range likely to be obtained in normal operation. It is intended for such use as automatically scaling a bar graph display.
•EngineeringUnits (optional): Specifies the units for the DataItem's value (e.g., DEGC, hertz, seconds).
