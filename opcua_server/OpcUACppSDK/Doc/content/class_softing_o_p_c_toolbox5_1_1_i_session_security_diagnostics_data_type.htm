virtual EnumStatusCode SoftingOPCToolbox5::ISessionSecurityDiagnosticsDataType::setClientUserIdHistory(const std::vector< tstring > & values) [pure virtual]

Sets a vector containing the name of the authenticated user currently active and the history of those names.

Each time the active user changes, an entry shall be made at the end of the list. The active user is always at the end of the list. Servers may restrict the size of this array, but shall support at least a size of 2.
