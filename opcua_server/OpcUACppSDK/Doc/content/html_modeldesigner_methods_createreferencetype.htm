CreateReferenceType

Top  Previous  Next

The CreateReferenceType method is used to create a new reference type.

Input Parameters:

BaseType

The BaseType has to be an existing ReferenceType node.

NodeId

The NodeId attribute of the node.

BrowseName

The BrowseName attribute of the node.

DisplayName

The DisplayName attribute of the node.

Description

The Description attribute of the node.

IsAbstract

The IsAbstract attribute of the node, describing whether this type can be instantiated.

Symmetric

The Symmetric attribute, describing whether the refefences have the same meaning in both directions.

InverseName

The InverseName attribute of the ReferenceType.
This attribute is mandatory for ReferenceTypes where the Symmetric attribute is set to false.

The attributes represent the most common attributes required to describe a ReferenceType.

Define a value for the attributes that shall be initialized with a specific value, otherwise set a NULL value or empty value to let the SDK generate a default value.

Some rarely required attributes are not listed by this method and can only be set via the method ModifyAttributes.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

NamespaceIndex

The index of the new created namespace, or if the namespace already existed before, the index of the existing one

NodeId

A NodeId with generated GUID identifier in the namespace with the highest namespace index.

 

Default attribute values:

Value
