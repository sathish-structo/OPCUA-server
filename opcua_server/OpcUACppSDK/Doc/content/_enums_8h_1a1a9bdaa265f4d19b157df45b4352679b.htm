EnumPubSubState Reference

Top  Previous  Next

Specifies the possible values of a PubSubState, which describes the possible states of the different PubSub components.

EnumPubSubState_Disabled

The PubSub component is configured but currently disabled.

EnumPubSubState_Paused

The PubSub component is enabled but currently paused by a parent component.

The parent component is either Disabled or Paused.

EnumPubSubState_Operational

The PubSub component is operational.

EnumPubSubState_Error

The PubSub component is in an error state.

EnumPubSubState_PreOperational

The PubSub component is in pre-operational state.
