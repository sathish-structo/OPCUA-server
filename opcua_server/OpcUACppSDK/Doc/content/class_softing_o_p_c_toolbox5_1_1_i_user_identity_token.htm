IUserIdentityToken Class Reference

Top  Previous  Next

#include "IUserIdentityToken.h"

 

The UserIdentityToken data type allows clients to specify the identity of the user they are acting on behalf of. The UserIdentityToken contains the credentials of the user associated with the client application.

The server uses these credentials to determine whether the client should be allowed to activate a Session and what resources the client may access during this session.

OPC UA defines the following types of identity tokens:

•IAnonymousIdentityToken for anonymous authentication
•IUserNameIdentityToken for username/password authentication
•IX509IdentityToken for certificate-based authentication
•IIssuedIdentityToken for authentication based on issued tokens (such as Kerberos tokens)
