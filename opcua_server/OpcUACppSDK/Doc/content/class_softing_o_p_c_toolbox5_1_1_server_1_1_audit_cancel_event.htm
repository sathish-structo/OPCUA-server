This event shall be generated if a client cancels requests.

The SourceName property for events of this type should be "Session/Cancel". The SourceNode property for events of this type should be assigned to the server object.

Beside mandatory properties of the base class additionally the following mandatory properties shall be set:

•RequestHandle: The requestHandle parameter of the Cancel Service call.
