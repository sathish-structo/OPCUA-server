The OPC Foundation has implemented GDS sample applications.

The sample GDS can be downloaded from https://github.com/OPCFoundation/UA-.NETStandard-Samples.

Follow the description regarding OPC UA Global Discovery Client and Global Discovery Server.

 

All GDS related samples provided by the OPC UA C++ Server & Client SDK V6.00 have to connect to an external Global Discovery Server.

Our implementation is tested using the sample provided by the OPC Foundation.

 

Note: This sample is just a sample how GDS functionality could be implemented. It does not (yet) support

•Validation of certificate expiry (whether update is needed)
•Certificate Push Management (is not implemented at all)
•...
