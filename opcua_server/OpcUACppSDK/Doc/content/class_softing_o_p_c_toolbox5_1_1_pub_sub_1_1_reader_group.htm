#include "PubSubReaderGroup.h"

 

The ReaderGroup class represents a PubSub ReaderGroup for publishing values.

A reader group is used to group a list of readers. It is not symmetric to a WriterGroup and it is not related to a particular NetworkMessage. The NetworkMessage related filter settings are on the DataSetReaders.

The configuration of the group (e.g. security) can be used as a default configuration of all contained DatSetReaders.
