virtual void SoftingOPCToolbox5::ReadProcessedDetails::setProcessingInterval(const OTDouble & value) [virtual]

Sets the processing interval (in ms) that divides the total range |StartTime - EndTime| into sub-ranges of the given length. The server shall return aggregate values for each sub-range. If the processing interval is zero, only one aggregate value shall be calculated for the total time range |StartTime - EndTime|.

virtual EnumStatusCode SoftingOPCToolbox5::ReadProcessedDetails::addAggregateType(const INodeId * value) [virtual]

Adds the NodeId of an aggregate to calculate.

The number of provided aggregate types must match the number of nodes to read (see Client::Session::historyReadProcessed()).
