CreateMethod

Top  Previous  Next

The CreateMethod method is used to create a new OPC UA method.

Input Parameters:

ParentNodeId

The NodeId of the parent node, where the new instance shall be created.

NodeId

The NodeId attribute of the node.

BrowseName

The BrowseName attribute of the node.

DisplayName

The DisplayName attribute of the node.

Description

The Description attribute of the node.

CreateInputArguments

A flag whether to create the InputArguments property for the new method. A not existing input arguments property means that the method has no input arguments. The NodeId of the property will be a generated GUID NodeId in the namespace of the method node. The initial content of the InputArguments property is an empty Argument array. Use the ModifyAttributes method to configure the InputArguments with the desired Argument array.

CreateOutputarguments

A flag whether to create the OutputArguments property for the new method. A not existing output arguments property means that the method has no output arguments. The NodeId of the property will be a generated GUID NodeId in the namespace of the method node. The initial content of the OutputArguments property is an empty Argument array. Use the ModifyAttributes method to configure the OutputArguments with the desired Argument array.

The attributes represent the most common attributes required to describe a Method.

Define a value for the attributes that shall be initialized with a specific value, otherwise set a NULL value or empty value to let the SDK generate a default value.

Some rarely required attributes are not listed by this method and can only be set via the method ModifyAttributes.

Output Parameters:

Status

This shows if the action was successful

Result

A list of errors or warnings occurred while executing the method, or a message that everything is fine.

NamespaceIndex

The index of the new created namespace, or if the namespace already existed before, the index of the existing one

MethodNodeId

The NodeId of the created Method

InputArgumentNodeId

The NodeId of the created input arguments property.
If the input argument CreateInputArguments is set to false, then output will be the NULL NodeId.

OutputArgumentNodeId

The NodeId of the created output arguments property.
If the input argument CreateOutputArguments is set to false, then output will be the NULL NodeId.

 

Default attribute values:

Attribute

Value

NodeId

A NodeId with generated GUID identifier in the namespace with the highest namespace index.

BrowseName

The namespace and the string representation of the NodeId attribute

DisplayName

The string representation of the NodeId

Description

Empty
