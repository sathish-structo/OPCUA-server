Trace Viewer

Top  Previous  Next

The OPC UA SDK supports remote tracing via TCP. This allows you to monitor trace messages coming from a SDK-based OPC UA server or client application. To display trace messages remotely, the SDK is shipped with a Trace Viewer application that is located in <InstallDir>/Tools. The following sections illustrate how to use this application:
