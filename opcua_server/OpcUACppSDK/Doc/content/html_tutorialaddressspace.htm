Design of the Information Model

Top  Previous  Next

One of the most important aspects in the development of OPC UA server applications is the design of the information model. The information model defines the way information is exposed to OPC UA clients. An information model is always represented by nodes in the address space of a server.

Knowledge of information models is also helpful for clients that want to interact with a server. For a client, an information model is a contract that defines the semantics of nodes. When a client knows about the information models that a server uses, it is able to interpret the meaning of nodes exposed in the address space.

An Information Model for Refrigerators

The picture below depicts a simple information model that is used to represent refrigerators in an OPC UA address space. Blue boxes indicate nodes that belong to the refrigerator model, grayed out nodes belong to OPC UA's base information model. Please refer to the article Notation of Nodes and References for reference on the notation of this diagram.

The refrigerator model introduces the RefrigeratorType.

The object type RefrigeratorType represents a refrigerator. It is merely a blue print for a specific refrigerator model. For each refrigerator the server wants to expose in the address space it creates a refrigerator object, i.e. an instance of the type RefrigeratorType.

The data points that are exposed by the refrigerator's proprietary interface are mapped to variables in the address space. Most of the variables in the refrigerator type are simple BaseDataVariables containing only a single value of a certain data type. Temperature variables are AnalogItems. AnalogItems expose three additional properties. The EURange describes the "normal" range of the variable's value. The InstrumentRange describes the allowed range of the variable's value. The EngineeringUnits property describes the engineering unit of the variable, in this case "degree Celsius".

The OPC UA Method "Open/Close door" provides a more convenient way to open or close the door of the refrigerator.
