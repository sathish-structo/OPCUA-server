#include "ServerAuditCertificateMismatchEvent.h"

 

This event type is generated if a certificate set of uses does not match the requested use for the certificate (i.e. application instance certificate, software certificate or CA).

This class does not define additional properties.

Constructs a new AuditCertificateMismatchEvent instance.

The newly created object has already the following properties initially set:

•Message: "Certificate mismatch" / "en"
•SourceName: "Security/Certificate"
•SourceNode: The server object
•ClientUserId: "System/OpenSecureChannel"
•Time: The actual time.
•ActionTimeStamp: The actual time.
•ReceiveTime: The actual time.
•Severity: 1
•ServerId: The ID of the server from the application description.
