const IReadValueId * SoftingOPCToolbox5::Server::ReadRequest::getValueId() const

Returns an object that describes the item and the attribute that is to be read.

For example, when a client wants to read the value of a variable, IReadValueId::getNodeId() returns the NodeId of the variable and IReadValueId::getAttributeId() returns 'EnumAttributeId_Value'.

const IDataValue * SoftingOPCToolbox5::Server::ReadRequest::getDataValue() const

Returns the value that is returned to the client. A server application has to use setDataValue() to provide the client the read value.

EnumStatusCode SoftingOPCToolbox5::Server::ReadRequest::setDiagnosticInfo(const IDiagnosticInfo * diagnosticInfo)

Sets vendor-specific diagnostic information for the read request. The diagnostic information can be omitted if no diagnostic information was encountered during processing of the request.

Parameters:

diagnosticInfo

The request-specific diagnostic information

Note:

Server applications should only return that kind of diagnostic information that has been requested by the client (see ReadTransaction::getDiagnosticsToReturn()).
