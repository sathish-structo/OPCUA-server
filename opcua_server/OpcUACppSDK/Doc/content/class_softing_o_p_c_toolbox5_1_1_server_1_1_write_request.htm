const IWriteValue * SoftingOPCToolbox5::Server::WriteRequest::getWriteValue()

Returns an object that describes the item and the attribute that is to be written.

For example, when a client wants to write the value of a variable, IWriteValue::getNodeId() returns the NodeId of the variable, IWriteValue::getAttributeId() returns 'EnumAttributeId_Value' and IWriteValue::getDataValue() returns the value that is to be written.

Note:

Please note that server applications need to verify whether the data type of the provided DataValue matches the data type of the attribute that is to be written.

ObjectPointer< Server::BaseNode > SoftingOPCToolbox5::Server::WriteRequest::getNode()

Returns the node whose attribute is to be written. This could be a variable or some other node type.

EnumStatusCode SoftingOPCToolbox5::Server::WriteRequest::setDiagnosticInfo(const IDiagnosticInfo * diagnosticInfo)

Sets vendor-specific diagnostic information for the Write request. The diagnostic information can be omitted if no diagnostic information was encountered during processing of the request.
