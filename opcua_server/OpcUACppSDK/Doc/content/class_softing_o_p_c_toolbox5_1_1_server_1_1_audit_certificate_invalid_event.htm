#include "ServerAuditCertificateInvalidEvent.h"

 

This event is generated if the data in the certificate do not match the given data.

It is generated if the HostName in the URL used to connect to the Server is not the same as one of the HostNames specified in the Certificate or if Application and Software Certificates contain an application or product URI that does not match the URI specified in the ApplicationDescription provided with the Certificate.

This class does not define any additional properties.
